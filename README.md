# AI Knowledge Platform 🚀

Backend system built using Spring Boot with focus on scalable architecture, cloud readiness, caching, and GenAI integration.

## 🔧 Tech Stack
- Java 17
- Spring Boot
- Maven
- PostgreSQL
- Spring Security (JWT)
- Redis (planned)
- Docker (planned)
- AWS / Azure (planned)

## ✨ Features
- REST APIs
- User Signup & Login (JWT Authentication)
- Secure endpoints
- File Upload (Local Storage)
- Scalable backend structure
- Cloud-ready design
